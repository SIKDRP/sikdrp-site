SIKDRP GitHub Pages Site

How to use:

1. Create a GitHub account.
2. Create a new repository named sikdrp-site.
3. Upload these files:
   - index.html
   - style.css
4. Go to repository Settings.
5. Click Pages.
6. Under Branch, choose main.
7. Click Save.
8. Your free site will go live at:

https://SIKDRP.github.io/sikdrp-site/

Edit your links:

Open index.html and replace href="#" for:
- eBay Store
- TikTok
- Etsy

Your Fourthwall link is already added.
